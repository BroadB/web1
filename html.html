HTML

<!doctype html> ; 이 문서가 html 언어로 작성되었음을 알림. 보통 맨 앞에 위치함.

<head> ~~~ </head> ; 웹페이지에 표시되지는 않지만 웹페이지에 대한 메타 데이터를 포함하는 것

<title> ~~~ </title> ; 웹페이지의 타이틀

<meta charset="utf-8"> ; 문서가 utf-8로 적혔음을 정의. 문자가 깨질 때 확인해보자

<body> ~~~ </body> ; 웹페이지의 본문

<ol> ~~~ </ol> ; ordered list. 목차의 부모 태그. 목차 앞에 숫자를 붙여준다

<ul> ~~~ </ul> ; unordered list. 목차의 부모 태그.

<li> ~~~ </li> ; 목차


<h1> ~~~ </h1> ; 헤드. h뒤에는 1부터 6까지 올 수 있다. 숫자가 커질 수록 중요도가 작으며 글씨 크기도 줄어든다.

<p> ~~~ </p> ; 한 단락
<p style="margin-top:45px;"> ~~~ </p> ; 단락 사이 공백이 45픽셀이 됨. css

<br> ; 줄 바꿈

<strong> ~~~</strong> ; 굵게

<u> ~~~ </u> ; 밑줄

<img src="이미지 경로"> ; 이미지 삽입
<img src="이미지 경로" width="n%"> ; 이미지 크기 조절

<iframe width="너비" height="높이" src="동영상 경로" frameborder="0" allowfullscreen> </iframe> ; 동영상 삽입
